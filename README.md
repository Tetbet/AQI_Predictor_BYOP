# AQI_Predictor_BYOP
India Air Quality Index Predictor and Forecaster for AI/ML Course.

## The Problem
Air pollution is a severe public health issue in India. Unpredictable spikes in AQI leave citizens unprepared. 

## The Solution
This project uses Machine Learning to:
1. **Predict** the exact AQI based on environmental pollutants (Regression).
2. **Classify** the air quality into health categories like 'Good' or 'Severe' (Classification).
3. **Forecast** future AQI trends based on historical data (Forecasting).
